## Heart Rate Zone Quick Reference

- **Z1**: 120-146 bpm (Recovery/Easy) - TARGET: 130-140 bpm for easy runs
- **Z2**: 146-154 bpm (Aerobic Base) - TARGET: 145-150 bpm for moderate easy runs
- **Z3**: 154-165 bpm (Tempo/Threshold) - TARGET: 158-165 bpm for tempo work
- **Z4**: 165-174 bpm (Lactate Threshold) - TARGET: 165-172 bpm for hill intervals
- **Z5**: 174-180 bpm (VO2 Max) - Not used in base building phase

**Trail Training Distribution** (80/15/5 Rule):
- **80%** = Z1-Z2 (easy running, foundation building)
- **15%** = Z3 (tempo work, sustained climbing efforts)
- **5%** = Z4+ (hill intervals, race-specific intensity)

**Compare to 10K Training** (60/30/10 Rule):
- **60%** = Z1-Z2 (easy running)
- **30%** = Z3-Z4 (tempo + intervals)
- **10%** = Z5 (VO2max work, race pace)

‚Üí Notice: Trail training is EASIER overall but LONGER duration
